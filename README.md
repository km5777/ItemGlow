The ultimate, highly-customizable item HUD for Minecraft.

Item Glow is a premium client-side mod that enhances your gameplay by providing a sleek, dynamic HUD element for whatever you're holding. No more opening your inventory just to check enchantment levels, durability, or food stats.

🌟 Key Features
- Dynamic Item Info: Instantly see item names, durability (bar & text), and stack sizes.
Intelligent Cycling: When holding items with multiple enchantments or potion effects, the HUD smoothly cycles through them one by one, showing you exactly what you have (e.g., Sharpness V • 1/3).
- Food & Potion Stats: View nutrition, saturation, and status effect durations directly on your HUD.
- Special Item Notes: Built-in support for "hidden" effects. Know exactly what a Golden Apple, Chorus Fruit, or Suspicious Stew will do before you eat it.
- Armor HUD & Peeking: A dedicated armor display with a "Peek" key (default: H) that reveals detailed enchantments on your equipped gear.
- Painting Details: Holding a painting? Item Glow will tell you its dimensions and variant name before you place it.
- Smooth Animations: Enjoy high-end visuals with customizable fade-ins, slide-ins, and item-switch transitions.

🛠️ Deep Customization Item Glow is designed to fit any playstyle. Using the Mod Menu and Cloth Config, you can tweak almost everything:
- Placement: Anchor the HUD to any corner of the screen or right above your hotbar.
- Aesthetics: Adjust scale, background opacity, accent colors, and text styles.
- Visibility: Set a custom fade delay or keep the HUD visible at all times.
- Toggle Elements: Individually enable/disable the icon, durability bar, enchantments, food previews, and more.

📋 Requirements
- Fabric Loader
- Fabric API
- Cloth Config API (Optional, for configuration)
- Mod Menu (Optional, for configuration)

🚀 Performance
- Item Glow is built with performance in mind. It is a client-only mod that uses efficient rendering techniques and data-component scraping to ensure zero impact on your frame rate.
